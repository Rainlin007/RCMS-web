# RCMS
## 功能
一个轻量的配置管理系统CMS，可用于APP的后台配置，比如升级配置，更换运营配置等。
## 结构
前端：React + antd

https://github.com/Rainlin007/RCMS-web

后端：Python3 + Django

https://github.com/Rainlin007/RCMS-Django

包括用户登录，project分组，json数据等
